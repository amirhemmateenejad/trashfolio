# Trashfolio Roadmap

This document outlines planned development phases.

---
## Phase 1 — MVP
Personal developer snippet manager.

Features:
- OTP authentication
- Projects
- Nested folders
- Snippets
- Tags
- Full‑text search
- REST API
- Vue SPA interface
- Dark mode
- Test coverage

Goal:

Deliver a stable personal snippet storage and retrieval system.

---

## Phase 2 — Sharing
Allow snippets to be shared between users.

Possible features:
- private sharing between users
- share via link
- snippet permissions
- collaboration on snippets

---

## Phase 3 — Public Snippets
Introduce public snippet discovery.

Possible features:
- public snippet pages
- public profile pages
- trending snippets
- public API access

## Future Ideas
These features are not part of MVP but may be implemented later.
- snippet version history
- favorites / pinned snippets
- import / export snippets
- keyboard command palette
- AI snippet suggestions
- tag suggestions
- collaboration
- syntax highlighting on server
